# Design Patterns
Design patterns are common architectural approaches that have been observed in software engineering. The notion of design pattern was popularized by Gang of Four book (1994).

## Design patterns

* Creational
  * Singleton
  * Builder
  * Factories
    * Abstract Factory
    * Factory Method
  * Prototype
* Structural
  * Adapter
  * Bridge
  * Composite
  * Decorator
  * Facade
  * Flyweight
  * Proxy
* Behavioral
  * Command
  * Chain of Responsibility
  * Interpreter
  * Iterator
  * Mediator
  * Memento
  * Null Object
  * Observer
  * State
  * Strategy
  * Template Method
  * Visitor
